# Security Policy

## Reporting Security Issues

Please do not open a public issue for security-sensitive reports.

If you believe this repository exposes a credential, signing key, device identity file, private dump, or other sensitive material, contact the maintainers privately through GitHub first. Include only the minimum detail needed to identify the affected file or behavior.

## Do Not Upload Sensitive Device Material

This project is public. Do not attach or paste any of the following in issues, pull requests, discussions, forks, or screenshots:

- Account credentials, API tokens, passwords, cookies, or session values.
- Android signing keys, keystores, `.pk8`, `.pem`, `.jks`, `.keystore`, or certificate private keys.
- Device identity files, NV partitions, modem/NVRAM dumps, serial numbers, IMEI/MEID values, Wi-Fi or Bluetooth MAC addresses, or provisioning data.
- Full ROM packages, APKs, proprietary vendor partitions, recovery dumps, generated flash images, or extracted proprietary blobs unless you have explicit redistribution rights.
- Personal data from the device owner, app accounts, messages, notifications, photos, or logs.

If a maintainer needs diagnostic output, redact it before sharing and keep the smallest useful excerpt.

## Supported Surface

This repository provides public documentation, integration scaffolding, and repo-local RaBobster handoff assets. It does not distribute a complete ROM, private signing material, private device backups, or proprietary Rabbit/MediaTek/CipherOS/LineageOS assets.
